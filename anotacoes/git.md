# Anotações de Git

## Comandos importantes

- `git status`: mostra o estado atual do repositório.
- `git diff`: mostra mudanças ainda não preparadas para commit.
- `git add`: prepara mudanças para o próximo commit.
- `git commit`: salva uma foto do projeto no histórico.
- `git pull`: baixa commits do repositório remoto e integra na branch atual.
- `git fetch`: atualiza as informações do remoto sem alterar os arquivos locais.
